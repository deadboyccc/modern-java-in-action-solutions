# study-notes

A minimal repository for documenting, studying, and breaking down selected concepts and exercises from core engineering texts. This is a personal sandbox focused on deliberate practice and conceptual clarity.

## 📚 Focus Areas

### Core Java (Vol 1 & 2)
Deep dive into JVM mechanics, the modularity system, advanced concurrency models, networking, and security primitives.  
Focus on low-level language behaviors and robust API design.

### Modern Java in Action
Functional programming mechanics, Stream pipelines, data transformations, and concurrency primitives.  
Focus on intermediate type mapping and pipeline visualization.

### Head First Design Patterns
Object-oriented design principles, behavioral, creational, and structural patterns.  
Focus on decoupling strategies and translating abstract patterns into tangible, clean architecture.

### Pragmatic Unit Testing in Java with JUnit 5 
Test-driven design, boundary conditions, mocking boundaries, and architectural testability.  
Focus on isolating components and writing maintainable, highly descriptive test suites.

### The Art of PostgreSQL
Declarative SQL patterns, analytical window functions, advanced aggregations, and relational modeling strategies.  
Focus on writing expressive, database-native logic.
